# Proyecto TransControl - Repositorio General

Bienvenido al repositorio general del proyecto **TransControl**. Este repositorio contiene todos los entregables, documentación y código fuente desarrollados durante el ciclo de vida del proyecto.

## 📂 Estructura del Repositorio

El repositorio está organizado en las siguientes carpetas para facilitar la revisión de cada etapa del proyecto:

* **`1.ELICITACION/`**  
  Contiene los documentos relacionados con el levantamiento y especificación de requerimientos del sistema.

* **`2.DiSEÑOS/`**  
  Incluye todos los modelos, diagramas de arquitectura, diseño de base de datos y maquetación de interfaces (UI/UX).

* **`3.CODIGO/`**  
  Contiene el código fuente completo de la aplicación. Aquí encontrarás el proyecto principal **TransControl**, dividido en:
  * `frontend`: Aplicación cliente desarrollada en React.
  * `backend`: Servidor y API desarrollados en Node.js con TypeScript.

* **`Analisis/`**  
  Documentación relacionada con el análisis del sistema, casos de uso y lógica de negocio.

* **`VIDEO/`**  
  Contiene el material audiovisual (o enlaces) con la demostración del funcionamiento del sistema.

---

## 🚀 Cómo ejecutar el proyecto (Código)

Si deseas probar el código fuente, dirígete a la carpeta `3.CODIGO/TransControl` y revisa el archivo `README.md` específico de esa carpeta, el cual contiene las instrucciones detalladas para instalar las dependencias y levantar los servidores locales de Frontend y Backend.
